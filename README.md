# DÚ: VERZOVACÍ SYSTÉMY: PRÁCE S GITEM A GITHUBEM

Tento repozitář slouží jako závěr domácího úkolu na git/GitHub. 

- Po dokončení všech částí úkolu proveďte fork tohoto repozitáře.
- Do souboru submit.txt přidejte nový řádek a dopište na něj "HOTOVO. Tým: [vypište členy svého týmu z části B]".
- Proveďte commit změn (a push, máte-li repozitář naklonován lokálně).
- Proveďte o pull request.
